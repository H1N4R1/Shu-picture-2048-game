# Shu-picture-2048-game
Java(Swing)初心者がにじさんじEN所属の闇ノシュウさんイラストで作った2048ゲームです。

##ゲームの遊び方
1.緑色のコードボタンをクリック
2.Zipをダウンロードという項目をクリック
3.ダウンロードをしたZipファイルを任意の場所で解凍
4. お使いのEclipseやVSCodeなどの開発アプリを開きます。
5. 解凍したフォルダ（Shu-picture-2048-game-main）を、そのままアプリの画面にドラッグ＆ドロップして読み込ませます。
6. フォルダの中にある「Picture2048Game.java」をクリックして開き、アプリの「実行（Run）」ボタンを押せばゲームが始まります！
